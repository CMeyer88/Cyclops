Main Navigation api documentation

The <main-nav> is a custom web component created for the CenturyLink Control Portal. It is a left-aligned bar that is the primary means of navigating between different sections of the Control Portal. Its contents are meant to be data-driven, as different users have varying levels of access to areas of the site.

Responsive behavior is built in, so at narrow viewports (portrait tablet or mobile device), the links will collapse into an icon-based menu. At wide viewports, the icon and label will be adjacent instead of stacked.

In the iframe below, the main-nav is the menu bar on the left.

Open Example Page 

    
<main-nav role="navigation" params="menus: menus, loading: loading, error: error, selectedItemId: selectedItemId"></main-nav>
    
    
$(function(){
  ko.applyBindings({
    menus: [...],
    self.loading = ko.observable(false);
    self.error = ko.observable(false);
    self.selectedItemId = ko.observable('infrastructure.manage');
  });
});
