Line Chart

A line chart is represented by a series of datapoints connected by a line. Line charts can be used to visualize data over a period of time.

		
$('#newLineChart').highcharts({
  chart: {
    type: 'line'
  },
  xAxis: {
    categories: ['Feb 14', 'Feb 15', 'Feb 16']
  },
  series: [{
    name: 'CPU',
    data: [5.7, 5.8, 5.9]
  }, {
    name: 'Memory',
    data: [5.9, 5.9, 6]
  }]
});
		
		
<div id="newLineChart" class="chart"></div>
