Account Switcher

The account-switcher allows a user to select an account in there available hierarchy to impersonate. All use should respect this impersonated account and present the user with a view as if they were logged in with a user that belonged to the account. This component should be included as part of the standard layout of your application.

Open Example Page 

    
<account-switcher params="accounts: accounts, currentAccountAlias: currentAccountAlias"></account-switcher>
    
    
$(function(){
  ko.applyBindings({
    accounts: [...],
    currentAccountAlias: '...'
  });
});
