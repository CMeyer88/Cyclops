Trackable Observable

Trackable Observable is an extension on the observable classes that tracks a committed value that can be reverted to. This is useful when creating edit forms that the user can either save or cancel out of.

When displaying trackableObservables you will want to bind to the committedValue in displays and just the default value for edits. 

  
var form = function(){
  var self = this;
  self.value = ko.trackableObservable("initial commited value");
  self.cancel = function(){
    // reset the value back to the currently committed value
    self.value.reset();
  };
  self.save = fucntion(){
    // commits the current working value
    self.value.commit();
  };
  return self;
};

Trackable Observable Array

Same thing as trackableObservable but instead of extending observable it extends observableArray.

  var self = this;
  self.array = ko.trackableObservableArray([]);

    self.array.reset();
  };
  self.save = fucntion(){
    // commits the current working value
